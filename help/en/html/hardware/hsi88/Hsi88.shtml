Supported Hardware

Computer Interfaces

JMRI supports the Hsi88 adapter to read out sensors on an S88 bus. For more information on the Hsi88 interface see the and the Hsi88 web site.

Limitations

JMRI currently supports the Hsi88 only when operated in Terminal Mode. The Hex Mode is not implemented as that protocal cannot be parsed withour look-ahead.

Connecting

Wiring

The Hsi88 comes in version for 9-pin serial interface or as a USB serial device.

Settings

  1. Make sure you have the latest version of JMRI. Start one of the JMRI applications.
  2. Go to the Connections tab of the JMRI Preferences panel. This opens automatically the first time a program is run, or you can select it from the "Edit" menu.
  3. Select "Hsi88" from the list as the System Manufacturer
  4. Next, select the proper serial port under Settings. Note that the USB Hsi88 drivers will appear as a serial port with OS specific names.
  5. Hsi88 reads sensors on the s88 bus. The length of the connected s88 sensor chains (in number of modules of 16 sensors) can be set for left, middle and right chain.
  6. Click "Save". You'll be asked if it's OK for the program to quit, click "Yes".
  7. Restart the program. You should be up and running.

Configuring Hsi88

The si88 automatically sets the Hsi88 hardware to Terminal Mode. All operation parameters (length of chains, terminal and hex mode, Hsi88 version) can be read and/or written Hsi88 Console.
